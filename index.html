Show differentiating instagram followers 0.1

There are 2 methods:

Using the Chrome/Edge extention (recommended)

  1. Download the zip HERE.
  2. Unzip the file and you should have a folder named compare-instagram-followers.
  3. In Chrome/Edge go to the extensions page (chrome://extensions or edge://extensions).
  4. Enable Developer Mode.
  5. Drag the compare-instagram-followers folder anywhere on the page to import it (do not delete the folder afterwards).
  6. Go to an Instagram profile.
  7. Click on the extention icon.

Using the JavaScript console script

  1. Click to copy the script to your clipboard.(If the copy does not work, press here)
  2. Open the JavaScript console (F12 or Ctrl+Shift+I).
  3. Paste the code (Ctrl+V) in the JavaScript console and press enter.
